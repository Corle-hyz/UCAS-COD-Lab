# ucas\_cod\_2020

## 描述:

本项目为**中国科学院大学(UCAS)**2020年春季学期**计算机组成原理实验课**作业

只上传了自己完成的代码, 不包括框架代码


## 内容:

- prj0: 加法器和累加器
- prj1: 算术逻辑单元ALU
- prj2: 单周期MIPS指令集CPU(44条指令)
- prj3: 内存及外设通路设计与处理器性能评估
- prj4: RISC-V指令集处理器实现
- prj5: 深度学习算法及硬件加速
